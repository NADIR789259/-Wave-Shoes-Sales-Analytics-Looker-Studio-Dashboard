# Wave Shoes Store Sales Dashboard

## Overview
An interactive sales analytics dashboard for Wave Shoes Store, built using **Looker Studio**. This project visualizes key business metrics across sales, cost, quantity, and profitability to enable data-driven decision-making.

---

## Key Metrics
| Metric | Value |
|---|---|
| Total Sales Amount | ₹113.33M |
| Total Cost | ₹71.23M |
| Quantity Sold | 24.05K |
| Total Transactions | 7.83K |
| Max Sales | ₹42.00K |

---

## Features
- **Sales Trend Analysis** — Month-over-month sales growth from ₹3.2M (Jan) to ₹113.3M (Dec)
- **Category Breakdown** — Quantity analysis across Casual, Formal, and Semi-Casual segments
- **Geographic Map** — Global sales distribution visualized on an interactive map
- **Product Mix (Donut Chart)** — Revenue share across Derby Shoes, Oxford Shoes, Brogues, Sneakers, Loafers, and Single Monk
- **State-wise Profit Table** — Top-performing states including Himachal Pradesh, Tripura, Sikkim, and more
- **Dynamic Filters** — Filter by Category, State, City, Order Type, and Customer Name

## Insights
- Sales grew consistently month-on-month, indicating strong and sustained market demand.
- Casual footwear dominates quantity sold compared to Formal and Semi-Casual segments.
- All six product categories hold nearly equal revenue share (~16.7% each), suggesting a balanced product portfolio.
- Himachal Pradesh leads state-wise profitability with ₹1,89,470 in profit.
